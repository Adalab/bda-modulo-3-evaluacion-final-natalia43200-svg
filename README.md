# BDA - Módulo 3: Evaluación Final

## Descripción

Este proyecto desarrolla un análisis exploratorio de datos sobre clientes de un programa de fidelización y su actividad de vuelos. El trabajo se organiza en tres fases: exploración y limpieza, análisis estadístico y visualización.

---

## Estructura del proyecto

### Fase 1: Exploración y limpieza
En esta fase se realizó una revisión inicial de los datos para comprender su estructura y calidad. Se analizaron:

- valores nulos
- duplicados
- valores únicos
- tipos de datos por columna

También se llevó a cabo la limpieza de datos, el tratamiento de variables y la unión de los datasets.

---

### Fase 2: Análisis estadístico
Se estudiaron las variables numéricas y categóricas del dataset unido.

#### 2.1 Variables numéricas
- estadísticas descriptivas
- identificación de valores atípicos
- análisis de correlación entre variables numéricas

#### 2.2 Variables categóricas
- análisis de frecuencias
- estudio de variables categóricas relevantes
- relaciones entre categorías

---

### Fase 3: Visualización
Se utilizaron gráficos para responder preguntas de negocio relacionadas con:

- vuelos reservados por mes
- relación entre distancia y puntos acumulados
- distribución de clientes por provincia
- salario promedio según nivel educativo
- tipos de tarjetas de fidelidad
- distribución por estado civil y género

---

## Archivos principales

- `Fase_1.ipynb`: exploración inicial y limpieza de datos

- `Fase_3.ipynb`: visualización de datos
